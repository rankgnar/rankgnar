## About

Technology consultant based in Stockholm. Self-taught, hands-on builder focused on AI systems, automation, and developer tooling. I design and ship products that solve real problems — from security auditing tools for the AI ecosystem to full-stack SaaS platforms.

I work across the entire stack: from infrastructure and CI/CD to frontend interfaces and LLM integrations. When I take on a project, I own it end to end.
